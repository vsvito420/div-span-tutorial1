CSS Box-Modell

Das CSS Box-Modell beschreibt, wie Elemente auf einer Webseite dargestellt werden. Jedes Element wird als rechteckige Box betrachtet, die aus vier Teilen besteht:

Margin (Außenabstand)
Border (Rahmen)
Padding (Innenabstand)
Content (Inhalt)
Der eigentliche Inhalt des Elements (Text, Bilder, etc.)

Die Bestandteile des Box-Modells

  • Content (Inhalt): Der eigentliche Inhalt des Elements, wie Text oder Bilder.
  • Padding (Innenabstand): Der Abstand zwischen dem Inhalt und dem Rahmen.
  • Border (Rahmen): Der Rahmen um das Element herum.
  • Margin (Außenabstand): Der Abstand zwischen dem Rahmen und anderen Elementen.

Beispiele für Box-Modell-Eigenschaften

Box 1
padding: 20px
border: 2px solid
margin: 10px
Box 2
padding: 30px 10px
border: 5px dashed
margin: 20px 5px
Box 3
padding: 10px 20px 30px 40px
border: 3px dotted
margin: 5px 15px 25px 10px

CSS-Syntax für Box-Modell-Eigenschaften

/* Einzelne Werte für alle Seiten */
padding: 20px;
margin: 10px;

/* Zwei Werte: oben/unten, links/rechts */
padding: 30px 10px;
margin: 20px 5px;

/* Vier Werte: oben, rechts, unten, links (im Uhrzeigersinn) */
padding: 10px 20px 30px 40px;
margin: 5px 15px 25px 10px;

/* Einzelne Seiten gezielt ansprechen */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;

Experimentiere mit dem Box-Modell

Box-Modell

Übungen zum Box-Modell

  1. Erstelle eine Box mit einem Inhalt deiner Wahl und experimentiere mit verschiedenen Padding-, Border- und Margin-Werten.
  2. Erstelle zwei nebeneinander liegende Boxen und passe die Margins an, um den Abstand zwischen ihnen zu kontrollieren.
  3. Erstelle eine zentrierte Box mit fester Breite, indem du margin: 0 auto; verwendest.
  4. Experimentiere mit verschiedenen Border-Stilen: solid, dashed, dotted, double.
